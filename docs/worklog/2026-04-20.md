## 二、搜尋引擎缺陷修正（knowledgeBase.ts）

### Bug 1：`hasTopicMatch` 縣市殘字問題

**根本原因**：函式僅移除縣市前綴（如「台北」），導致「台北市公安申報...」→「 市公安申報...」，CJK 塊以「市」開頭，無法匹配任何條目關鍵字或使用正式詞「公共安全」的答案文字。

**修正方式**：在移除前綴之前，先移除完整縣市名（如「台北市」→空格），消除「市」殘字。

```typescript
// 先移除完整縣市名（如「台北市」），避免僅移除前綴後留下「市」殘字
topicQ = topicQ.split(queryCounty.toLowerCase()).join(' ')
```

**影響**：Q11（罰鍰）、Q14（現場照片）、Q19（申報頻率）從「無結果/錯誤條目」→ 精準命中。

### Bug 2：`expandQuerySynonyms` 新增口語→正式詞映射

**問題**：使用者說「公安申報」，知識庫條目使用正式詞「公共安全申報」，文字比對無法匹配。

**修正**：新增同義詞展開規則：

```typescript
if (/公安申報|公安檢查/.test(q)) extra += ' 公共安全'
```
