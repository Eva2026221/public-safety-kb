# 解釋函文閱讀規則

> 適用對象：公安檢查從業人員、知識庫維護人員
> 資料來源：本庫 1–14 號解釋函文件（docx）

---

## 一、函文固定結構

每筆解釋函由以下欄位組成，閱讀時先定位各欄位再看內容：

```
主旨（標題）        → 說明所詢疑義，結尾為「乙案」或「一案」
發文機關            → 如「建築管理組」
發布日期            → 西元年格式（YYYY-MM-DD）
文號                → 民國年格式（見下節）
正文                → 先引法條，再解釋疑義，分點論述
最後更新日期        → 可能晚於發布日期（代表曾補充修正）
```

---

## 二、文號解讀

格式：`[民國年].[月].[日][機關代號]字第[流水號]號函`

範例：`103.3.3 營署建管字第1030009253號函`

| 欄位 | 說明 |
|------|------|
| `103` | 民國103年（= 西元2014年，換算：民國年 + 1911） |
| `3.3` | 3月3日 |
| `營署建管字` | 機關業務代號（內政部營建署建管組） |
| `1030009253` | 流水號，前3碼通常為民國年 |
| `號函` / `號令` | 「函」= 行政解釋；「令」= 法規命令，效力較高 |

---

## 三、正文閱讀順序

### 3-1 先看「合先敘明」段

正文通常先引用法條全文作為前提（常用語：「按…規定」、「…業有明定」、「合先敘明」）。這段確認本函適用的法規版本及條號。

### 3-2 找結論位置

| 正文形式 | 結論位置 |
|----------|----------|
| 分點論述（一、二、三、） | **最後一點**通常是結論 |
| 單段敘述 | 末句或「爰此」、「是以」之後 |
| 附會議紀錄 | 「結論：」欄位 |

### 3-3 注意個案但書

出現以下語句代表**不能一概而論**，須就個案認定：

- 「案涉個案事實認定，請本於權責核處」
- 「宜個案認之」
- 「請依上開規定，本於權責，認定核處」

## 四、法令適用時間點（最關鍵）

依中央法規標準法第18條，建築物適用**行為時之法令**：

| 行為 | 適用法令時間點 |
|------|----------------|
| 建造執照申請 | 申請當時之建築技術規則 |
| 變更使用執照申請 | 申請變更當時之法令 |
| 公安申報檢查 | 建造（或最後一次合法變更）當時之法令 |

實務口訣：**「用照年份定規定，新增行為用新法」**

---

## 五、新舊函示效力判斷

同一主題可能有多個不同年份的函示：

1. **新函明確廢止舊函**：依新函。
2. **新函補充舊函**：兩者並存，新函補充說明例外情形。
3. **新函未提及舊函**：原則上新函優先，但舊函針對特定情形之解釋仍有效。
4. **最後更新日期 ≠ 發布日期**：更新日期代表本庫彙整時確認仍有效，或做了文字補充。

查詢時建議：同主題函示**依發布日期從新到舊**排序閱讀，以最新函示為主。

## 六、中央函示 vs. 地方規定

| 層級 | 來源 | 效力 |
|------|------|------|
| 中央 | 內政部、內政部營建署 | 全國適用 |
| 直轄市/縣市 | 台北市、新北市、台中市… | 僅該轄區適用 |

**同一問題若有地方函示，地方函示優先於中央函示（在該縣市）**，但地方規定不得低於中央最低標準。

---

## 七、常用法規引用對照

| 法規名稱（簡稱） | 全名 |
|-----------------|------|
| 設計施工編第XX條 | 建築技術規則建築設計施工編第XX條 |
| 建築法第77條第3項 | 建築物公共安全定期申報義務 |
| 申報辦法 | 建築物公共安全檢查簽證及申報辦法 |
| 使用類組辦法 | 建築物使用類組及變更使用辦法 |

> 注意：條號可能因法規修正而改變。函文內若有備註（如「第16款現已修正為第19款」），以備註為準。

## 八、特殊文型辨識

| 文型 | 特徵 | 閱讀重點 |
|------|------|----------|
| 函（行政解釋） | 「號函」結尾 | 解釋疑義，非強制規範 |
| 令（法規命令） | 「號令」結尾 | 具法規效力，強制適用 |
| 書函 | 「號書函」 | 個案回覆，參考用 |
| 會議紀錄 | 「研商…會議紀錄」 | 看「結論」欄，決議有行政拘束力 |

---

## 九、快速閱讀流程（SOP）

```
① 看主旨 → 確認問題主題與檢查項目（防火區劃/安全梯…）
② 看文號 → 換算年份，判斷是否為近期函示
③ 看機關 → 中央或地方？影響適用範圍
④ 看「合先敘明」→ 確認引用的法條與版本
⑤ 找結論 → 最後一點或「爰此」之後
⑥ 確認個案但書 → 有無「本於權責」語句
⑦ 比較新舊 → 同主題是否有更新的函示
```
